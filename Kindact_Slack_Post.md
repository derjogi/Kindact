Hi all — I've been working on a concept called **Kindact** and would really appreciate some eyes and honest feedback on it.

The core idea: a voluntary, open-source platform that sits *alongside* existing governments as an incentive layer for collective action. Anyone can raise issues, communities deliberate using bias-reducing mechanisms (anonymized discussion, structured arguments, AI-assisted synthesis), decisions are made through fluid voting with liquid delegation, and — here's the key part — **new tokens are minted only when community-approved work is verifiably implemented**. So the economic system directly ties doing good to receiving value, rather than relying on taxes or donations upfront.

It's not trying to replace anything. It's trying to create infrastructure where solving problems together is actually rewarded.

I wrote a longer post that goes into the mechanics — deliberation design, fraud prevention, token economics, how it handles things like the cold-start problem and Moloch traps: [link to blog post]

This is very much a vision in progress, not a finished product. I'd genuinely welcome critique — what's naive, what's missing, what wouldn't work. Especially interested in perspectives from people thinking about governance, economics, or regenerative systems.
