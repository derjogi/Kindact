A — Subtle Left Border + Label

Thin colored left border (green = supporting, orange = counter). Small pill label. Neutral comments have no decoration. Gentle — doesn't dominate the visual.

B — Background Tint + Text Label

Entire comment gets a very faint background tint (green/orange). Small text label instead of pill. More visible stance at a glance, but could feel busy with many comments.

C — Icon-only Minimal

Just a small colored arrow icon next to the author name. No background change, no pill. Most minimal — stance is there if you look, but doesn't distract from reading the actual discussion.

Design note

The stance (supporting/counter/neutral) is set by the comment author when posting — it describes their reply's relationship to the parent. It's optional: if you're just asking a question or chatting, you leave it unset (neutral). The terminology "supporting/counter" is relative to the parent comment, not the overall issue.
